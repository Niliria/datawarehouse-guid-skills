# 数据库元数据导出器

本技能将数据库表元数据导出到Excel文件，包括表结构、外键关系和数据质量的详细信息，帮助用户全面了解数据库结构。

## 核心功能

- **导出所有表元数据**到单个Excel工作表
- **包含字段注释**和自动生成的注释填充
- **生成表中文名**，自动解析表名生成中文名称
- **计算字段空值率**以评估数据质量
- **解析外键引用关系**，显示引用的表和字段
- **支持多种数据库类型**（MySQL、SQLite、PostgreSQL、Oracle、SQL Server）
- **生成格式化的Excel文件**，包含适当的样式和冻结表头

## 使用方法

### 前置条件

- Python 3.x
- 所需库：`pyyaml`、`pymysql`、`openpyxl`
- 数据库配置文件：`Input/config.yaml`

### 执行命令

#### 导出所有表元数据

```bash
python3 .claude/skills/metadata_parse/scripts/step1-ddl-parse/export_all_tables.py
```

## 输出文件

- **`Output/all_tables_metadata.xlsx`**：所有表元数据在单个工作表中，包含完整的字段信息和外键关系

## Excel文件结构

| 列名      | 描述                   |
| ------- | -------------------- |
| 数据源类型   | 数据库类型（例如：mysql）      |
| 业务数据库名称 | 数据库名称                |
| 表名      | 表名称                  |
| 表中文名    | 表的中文名称               |
| 字段名     | 字段名称                 |
| 字段注释    | 数据库中定义的字段注释          |
| 字段注释填充  | 自动生成的字段注释（当数据库中无注释时） |
| 数据类型    | 字段数据类型               |
| 是否为空    | 字段是否可为空（YES/NO）      |
| 默认值     | 字段默认值                |
| 主键      | 是否为主键（YES/NO）        |
| 外键      | 是否为外键（YES/NO）        |
| 外键引用    | 外键引用的表和字段            |
| 字段空值率   | 字段空值占比百分比            |

## 高级特性

1. **智能字段名拆解**：支持下划线分隔、驼峰命名和连续小写字母的字段名拆解
2. **字段注释自动生成**：当数据库中没有字段注释时，会自动根据字段名拆解和翻译生成中文注释
3. **表中文名生成**：自动解析表名生成中文名称，提高表结构的可读性
4. **词汇映射自定义**：可通过 `Input/word_map_reference.xlsx` 自定义词汇翻译
5. **外键引用解析**：自动识别并解析外键引用关系，显示引用的表和字段
6. **数据质量评估**：通过字段空值率列评估数据完整性
7. **表关系分析**：通过外键引用信息了解表之间的关联关系

## 应用场景

1. **数据库结构分析**：全面了解所有表的详细结构和字段信息
2. **表关系分析**：通过外键引用信息了解表之间的关联关系
3. **数据质量评估**：识别空值率高的字段，评估数据完整性
4. **文档生成**：创建标准化的数据库结构Excel文档
5. **迁移规划**：在数据库迁移前分析表结构和字段特性
6. **数据模型设计**：基于现有表结构分析，优化数据模型设计
7. **开发参考**：为开发人员提供完整的数据库结构参考文档
8. **代码生成**：利用元数据和外键关系生成ORM模型或数据访问层代码
