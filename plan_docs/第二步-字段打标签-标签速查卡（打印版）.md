# 字段打标签标签速查卡（打印版）

> 打印出来贴在显示器旁边，标注时快速查阅。

---

## 一、核心标签（8 个）

| 标签 | 符号 | 判定关键词 | 示例 |
|------|------|-----------|------|
| [业务键] | 🔑 | `_id`, `_code`, `_no`, `_num` | order_id, product_code |
| [代理键] | 🆔 | `id`（自增）, `sk_` | id, sk_user_id |
| [外键] | 🔗 | 引用其他表的 ID | user_id → ods_user |
| [退化维度] | 📛 | 低基数枚举 | order_status, payment_channel |
| [可加度量] | ➕ | amount, qty, count | order_amount, quantity |
| [半可加度量] | ➕⏳ | balance, stock | account_balance |
| [不可加度量] | ❌ | ratio, rate, avg | conversion_rate |
| [维度属性] | 📝 | name, category, type | user_name, product_category |
| [业务时间] | 🕐 | create_time, pay_time | order_create_time |
| [技术时间] | ⏱️ | update_time, etl_time | update_time |
| [技术字段] | 🛠️ | is_deleted, version | is_deleted |

---

## 二、字段类型 → 标签映射

### 数值型（int/decimal/float）

```
金额/数量/次数 → ➕ [可加度量]
余额/库存 → ➕⏳ [半可加度量]
比率/百分比 → ❌ [不可加度量]
```

### 时间型（datetime/timestamp）

```
create_time/pay_time → 🕐 [业务时间]
update_time/etl_time → ⏱️ [技术时间]
```

### 文本型（varchar/char/text）

```
含 _id/_code/_no → 🔑 [业务键]
描述性（name/category）→ 📝 [维度属性]
低基数枚举 → 📛 [退化维度]
```

### 整数型 + 自增

```
id (AUTO_INCREMENT) → 🆔 [代理键]
```

## 四、判定流程图

```
字段是什么类型？
├─ 数值型 → 金额/数量？→ ➕
│          余额/库存？→ ➕⏳
│          比率/百分比？→ ❌
│
├─ 时间型 → create/pay/ship？→ 🕐
│          update/etl？→ ⏱️
│
├─ 文本型 → 含 _id/_code？→ 🔑
│          描述性？→ 📝
│          低基数？→ 📛
│
└─ 整数+自增 → 🆔
```

## 六、快速检查清单

标注完一张表后，快速检查：

- [ ] 所有 ID 都标了 🔑 或 🆔？
- [ ] 所有外键都标了 🔗 并注明引用表？
- [ ] 所有数值都标了 ➕ / ➕⏳ / ❌？
- [ ] 所有时间都区分了 🕐 和 ⏱️？
- [ ] 所有文本都标了 📝 或 📛？
- [ ] 所有系统字段都标了 🛠️？

---

## 七、打印建议

1. **纸张大小：** A5（对折后可放显示器旁边）
2. **字体：** 宋体小五号
3. **重点加粗：** 标签符号（🔑🔗📝 等）
4. **裁剪线：** 沿虚线裁剪，方便张贴
